# 轻量化量化回测引擎

## 项目背景
为解决传统量化开发中手动搭建回测框架繁琐、调试效率低的问题，基于Python开发了这套独立通用回测引擎。
使用ai辅助梳理架构、编写代码、调试运行逻辑，大幅降低了量化工具开发的门槛。

## 核心功能模块
1.  **公开行情数据获取**
    - 支持从东方财富、腾讯财经等公开渠道获取免费行情数据
    - 仅包含基础爬虫能力，无任何破解/反爬逻辑，合规获取公开数据

2.  **通用回测引擎**
    - 标准化回测运行流程，支持多周期、多品种回测
    - 基础仓位管理与交易模拟能力

3.  **绩效指标统计**
    - 自动计算收益、最大回撤、夏普比率等核心指标
    - 回测结果结构化输出，便于分析优化

4.  **可扩展策略接口**
    - 框架支持加载自定义策略文件，可快速接入不同交易逻辑
    - 本次开源仅提供引擎主体，不含任何自定义策略代码

## 重要声明（必看）
- 本项目为**纯底层工具框架**，不包含任何实战交易模型、自研因子或核心买卖信号。
